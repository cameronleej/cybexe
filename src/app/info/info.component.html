Malware Types

About: A useful or seemingly useful program that contains hidden code of a malicious nature that executes when the program is invoked.
Capabilities:
• Delete, modify, or steal data
• Spy on users
• Breach networks
• Launch DDoS attacks
• Remotely control devices
Real World Example: 2011 - ZeuS

About: A program designed to block access to a computer system until a sum of money is paid.
Capabilities:
• Hold devices hostage
• Make data inaccessible
• Result in financial loss
Real World Example: 2017 - WannaCry

About: A program that replicates itself by attaching to other programs or files, where it hides until activated. Requires a "host" to live.
Capabilities:
• Steal data
• Send more viruses to contacts
• Launch DDoS attacks
• Launch ransomware attacks
Real World Example: 2000 - ILOVEYOU virus

About: A self-replicating program that propagates itself through a network onto other computer systems without requiring a host program or any user intervention to replicate.
Capabilities:
• Delete, modify, or steal data
• Install backdoors
• Launch DDoS attacks
• Launch ransomware attacks
• Infect many devices at once
Real World Example: 2003 - SQL Slammer

  • Every 11 seconds, a business falls victim to a ransomware attack.
  • A total internet shutdown would lead to a GDP loss of 1.9% daily for a country with high connectivity and a daily loss of 0.4% for a low-connectivity country.
